## Target Audience

- **Primary**: 技术宅/打工人 — 正在用 Claude Code 做项目（代码或非代码），不懂 Git
- **Secondary**: 内容创作者 — 用 AI 辅助写作、管理配置文件
- **Tertiary**: AI 新手 — 刚开始用 AI agent，需要安全网

## Value Proposition

**为什么用户要看？**
1. **痛点共鸣**: 6 个真实场景（CLAUDE.md 改坏、版本找不回、多人协作灾难...）
2. **实用价值**: 只需 6 个命令，门槛极低
3. **安全感**: "AI 帮你改坏了也不怕"
4. **收藏理由**: Git 命令速查，需要时回来看

- **收藏价值**: 极高 — 6 个命令 + 自动存档配置 = 实用速查卡

## Key Points Extraction

1. **开头故事**: Claude 改坏文章 → 回不去了（痛点）
2. **6 个痛点场景**: CLAUDE.md 改坏 / 版本丢失 / 多人协作 / v1v2v3 命名 / 风格快照丢失 / Claude 偷改配置
3. **Git = 时光机 + 平行宇宙**（核心比喻）
4. **6 个基础命令**: init / add+commit / log / diff / checkout / branch
5. **自动存档**: CLAUDE.md 里加一条规则
6. **总结**: Git 是所有用 AI 做项目的人的安全网
