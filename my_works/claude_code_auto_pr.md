最近正在搭 OpenClaw，我能想到的使用场景，就是让它帮我处理我正在开发的项目的 PR。

因为我在独立开发，写一个软件的过程中，常常会发现有一些问题，甚至是一些 bug，我没有时间去处理。有的时候，一些问题又可能会在不经意间冒出来。就像地上的口香糖一样让人觉得难受。

最近我尝试把这些问题都写成 issue，但是我的确也没有时间去处理。

所以，我最近一直在搭建处理 issue 然后提 PR 的工作流，想着之后能不能通过 OpenClaw 这样的工具，让 AI 每天晚上自动帮我处理，第二天统一 review 合入即可。

结果，我 OpenClaw 还没搭建好，Claude Code 就推出这个功能了。

快要用起来 🫠
